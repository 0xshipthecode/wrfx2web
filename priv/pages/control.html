This is a prototype of a online coupled fire-weather simulator submitted for the 2013 Oct 1 review deadline of the NASA grant 11-FIRES11-0038. It provides the capability to start, monitor and stop a wildland fire simulation using the coupled weather/fire code WRF-Fire.
